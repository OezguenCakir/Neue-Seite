Was DataCamp bietet

DataCamp bietet - hauptsächlich zu Python, R und SQL - eine Vielzahl von Kursen an. Zertifizierrn kann man sich als Data Scientist und Analyst. Letztere Zertifizierung habe ich absolviert. Beide Zertifizierungen kann man wahlweise mit SQL und Python oder R absolvieren.

Mit Python lernt man Datensätze zu importieren, bereinigen, bearbeiten und visualisieren. Hauptsächlich arbeitete ich mit den Libraries Pandas, NumPy, Matplotlib und Seaborn. Für SQL lernt man die grundsätzliche Syntax, Abfragen, Joinen von Tabellen, Window Functions, CTEs und vieles Mehr. Vor allem die SQL-Inhalte bei DataCamp kann ich loben.

Ich hätte lieber R statt Python wählen sollen

Da ich ursprünglich den Weg zum Data Scientist gehen wollte und in der Universität zuletzt mit Python arbeitete, habe ich Python gewählt. Ich habe aber gemerkt, dass die Arbeit als Data Analyst mehr zu mir passt und dadurch, dass ich auch mehr im Studium mit R arbeitete, würde ich im Nachhinein eher R wählen. Ich würde sogar sagen, dass ich es bereue Python gewählt zu haben, da ich dafür keinen Zweck sehe, wo ich lieber auf Python zugreifen würde als auf Excel, R oder andere Mittel.

Inhalte haben Schwerpunkt in Code-Aufgaben

Die Kurse sind sehr interaktiv. Kapitel beginnen mit Videos, wobei der Schwerpunkt auf den Aufgaben liegt. DataCamp bietet direkt eine Konsole für den einzugebenden Code. Kritikpunkt ist, dass die vorausgefüllten Teile teilweise zu hilfreich sind und das Abrufen der Tipps und Lösungen nicht genug „bestraft“ wird. Der Schwerpunkt der Inhalte liegt auf Code und weniger auf theoretische Inhalte.

Die Zertifizierung ist von allen drei von mir absolvierten Kursen deutlich am anstrengendsten. Die Tests bezüglich Python waren verhältnismäßig einfach, während die zu SQL dahingegen schwieriger waren. Die Case Study live und in Englisch zu präsentieren, war spannend und herausfordernd.
